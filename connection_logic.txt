Step 1: Check if already connected (status='accepted')
├─ YES → Display: "You are already connected with this user."
│        └─ Return to Main Menu
└─ NO → Continue

Step 2: Check if they have pending request to you
├─ YES → Display: "[User Name] already sent you a request!"
│        Prompt: "Would you like to accept it? (Y/N): "
│        ├─ Y → UPDATE status to 'accepted'
│        │      Display: "Connection accepted!"
│        │      └─ Return to Main Menu
│        └─ N → Display: "Request not accepted."
│               └─ Return to Main Menu
└─ NO → Continue

Step 3: Check if you have pending request to them
├─ YES → Display: "You already sent a connection request to this user."
│        └─ Return to Main Menu
└─ NO → Continue

Step 4: Check if there's a declined request
├─ YES → UPDATE status from 'declined' to 'pending'
│        Display: "Connection request sent successfully!"
│        └─ Return to Main Menu
└─ NO → Continue

Step 5: No existing connection found
└─ INSERT new request (status = 'pending')
   Display: "Connection request sent successfully!"
   └─ Return to Main Menu
